## Built with:
- NextJS
- React Hooks
- Typescript
- Styled Components
- MongoDB
- React Google Maps

## Requirements
- EditorConfig for VSCode 
- node v12.x (use nvm)
- yarn (install via npm)
- API key from React Google Maps
